## [Unreleased]

## [1.0.0] - 2026-07-23

### Added

- Initial repository foundation for authoring, validating, testing, and publishing portable Agent Skills.
- Acta 0.1.0 canonical design, artifact protocol, deterministic materializer, and static HTML validation.
- Fifteen stable development skills with trigger/non-trigger cases and optional Codex presentation sidecars.
- Immutable Acta design provenance, public core-cycle fixture, and versioned manual promotion report.

### Changed

- Whole-bundle installation is now the primary product path while isolated skills remain supported.
- The initial fifteen skills were promoted together after passing Codex and Claude Code activation and full core-cycle evaluation.
- Acta v2 `0.2.0` is the authoritative offline instrument/record system; the rejected remote-API expansion remains excluded.

No `v0.x` release was published. The first release is `v1.0.0`; ADR 0005 records the explicit deferral of browser and accessibility evidence for this initial promotion.
