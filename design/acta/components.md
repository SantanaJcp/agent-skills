# Acta component contracts

Acta is a print-first engineering record. Its stable shell is ProvenanceHeader → ContextHeader → optional SourcePrompt → SummaryBand → SectionNavigator → sections → ExportBar → sources → colophon.

Required contracts: ArtifactShell, ProvenanceHeader, ContextHeader, SourcePrompt, SummaryBand, SectionNavigator, Disclosure, StatusBadge, MetricStrip, EvidenceBlock, DecisionCard, DecisionGate, ComparisonSet, CodeSurface, DataTable, Timeline, StickyRail, DiagramNode/Edge, EditorControl, ExportBar, StatusRegion, Quiz, and NoScriptFallback.

Use serif type for reading, monospace for records, and system UI only for controls. Use type, spacing, rules, and tint for hierarchy; use no shadows. Label load-bearing claims Fact, Inference, or Simplification. Number decisions D-01 onward. Show equal-fidelity alternatives before a recommendation. Never fold a warning, gate, or open-decision evidence. Every diagram needs a textual equivalent. Every interaction must end in text.
