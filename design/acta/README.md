# Acta authoring source

Acta 0.1.0 is the canonical publisher-only design and artifact source for the fifteen-skill development suite. It is materialized into self-contained installed references; consumers never load this directory at runtime.

The immutable design provenance and visual baseline live under the repository design documentation. Run `npm run acta` after changing tokens, behavior, protocol, components, or recipes, then run `npm run check`.

Acta adapts principles from Thariq Shihipar's Apache-2.0 HTML-effectiveness project at commit `1787245d94aa680edf18b52027e3f859032776ba`. The Acta identity, token system, component grammar, and recipes are original.
